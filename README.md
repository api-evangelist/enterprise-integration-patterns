# Enterprise Integration Patterns (enterprise-integration-patterns)

Enterprise Integration Patterns (EIP) is a catalog of architectural design patterns for integrating enterprise applications. The patterns describe messaging-based integration scenarios and provide a common vocabulary for architects and developers to design and implement integration solutions. The canonical reference is the book by Gregor Hohpe and Bobby Woolf, organized into six pattern categories: Messaging Channels, Message Construction, Message Routing, Message Transformation, Messaging Endpoints, and System Management.
